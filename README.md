## Visão Geral

Esse projeto serve como um **mini sistema de cadastro** para pessoas, com comandos no terminal. É ideal para fins educativos ou para praticar conceitos de manipulação de dados, entrada de usuário, e validações simples.

---

## Funcionalidades

- Cadastrar uma pessoa com os seguintes dados:
  - Nome  
  - Idade  
  - CPF  
  - E-mail  
  - Senha  
- Validar:
  - Formato de e-mail  
  - Tamanho mínimo da senha  
  - (Opcional) Formato/função para validar CPF  
- Listar todas as pessoas cadastradas na sessão atual  
- Interação simples via menu no terminal

---

## Tecnologias Utilizadas

- **Python 3.13 — Linguagem principal   
- armazenamento em memória (lista ou dicionários) — sem banco de dados externo
